

Project Name : PetCare

Purpose:
This is a React web application for managing winter care tips and services.

Live URL:
(https://petcarea9.netlify.app/)

## Key Features

1. Allows users to log in and manage their profiles.
2.  Provides useful information and advice for winter care.
3.  Displays various services and details in card format.
Responsive Design: Optimized for both desktop and mobile devices.

## NPM Packages Used

react,firebase,reactrouter
